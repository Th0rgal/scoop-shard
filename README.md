# Scoop Bucket for Shard

This is the official [Scoop](https://scoop.sh) bucket for [Shard](https://shard.thomas.md), a minimal, content-addressed Minecraft launcher.

## Installation

```powershell
# Add the bucket
scoop bucket add shard https://github.com/th0rgal/scoop-shard

# Install the CLI
scoop install shard
```

## Updating

```powershell
scoop update shard
```

## Uninstalling

```powershell
scoop uninstall shard

# Optionally remove the bucket
scoop bucket rm shard
```

## Available Packages

| Package | Description |
|---------|-------------|
| `shard` | Shard CLI for managing Minecraft profiles |
